LatLng
==========

The LatLng object is needed for creating coordinates with latitude and longitude.

Constructor
-----------

```LatLng($lat, $lng)```

Properties
----------

**lat**: float

Latitude coordinate.

**lng**: float

Longitude coordinate.

Examples
--------
Basic Example
```php
use WildSiena\GoogleMapsBundle\Model\LatLng;

new LatLng(lat: 42.42, lng:42.42);
```
